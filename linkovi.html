Diskpart (Boot media)

- diskpart
- select disk <#> (USB flash disk letter)
- Clean
- Create partition primary
- Select partition 1
- Active
- Format quick fs=FAT32 quick
- assign
- exit

Diskpart (HDD)

- diskpart
- select disk 0
- clean
- convert gpt
- create partition efi size=300
- assign letter=k
- format quick fs=FAT32
- create partition msr size=128
- create partition primary
- assign letter=c
- format quick fs=NTFS
- exit
